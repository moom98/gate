# iOS Client

SwiftUI application for iOS to manage Claude Code permission requests.

This will be implemented in `feat/070-ios-minimal`.

## Requirements
- iOS 15.0+
- Xcode 14+
- Same Wi-Fi network as broker

## Features (Planned)
- Manual IP:port connection configuration
- WebSocket real-time updates
- Allow/Deny buttons for permission requests
- Token-based authentication
- Foreground-only operation
